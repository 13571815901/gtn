GTN
---

Python bindings to `GTN <https://github.com/facebookresearch/gtn>`_, a library
for automatic differentiation with weighted finite-state transducers (WFSTs).
Visit the `C++ documentation <https://gtn.readthedocs.io/>`_ for a more
an in-depth introduction.
